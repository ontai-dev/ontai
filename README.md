# ontai

The ONT Platform monorepo root. Operator Native Thinking.

**Website:** https://ontai.dev
**Schema:** https://schema.ontai.dev
**Founding Document:** https://ontai.dev/docs/index.md

## What Is Here

This repository contains the Governor constitution, session
state documents, and the ontai.dev website served via GitHub
Pages from the docs/ directory.

## Operator Repositories

| Repo | Description |
|------|-------------|
| [guardian](https://github.com/ontai-dev/guardian) | Trust root and security substrate |
| [platform](https://github.com/ontai-dev/platform) | Cluster lifecycle authority |
| [wrapper](https://github.com/ontai-dev/wrapper) | Pack delivery engine |
| [conductor](https://github.com/ontai-dev/conductor) | Execution intelligence |
| [seam-core](https://github.com/ontai-dev/seam-core) | Lineage and semantic DNS |
| [domain-core](https://github.com/ontai-dev/domain-core) | Domain primitive declarations |
| [app-core](https://github.com/ontai-dev/app-core) | Application domain primitives |
| [ontai-schema](https://github.com/ontai-dev/ontai-schema) | OpenAPI JSON Schema specification |

## Documentation

The docs/ directory is served as the ontai.dev website via
GitHub Pages. The founding document is at docs/index.md.
